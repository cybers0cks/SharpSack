# SharpSack
#### Random sack of C# tools

```
SharpSack.exe help

Print help                                    Example: SharpSack.exe help
Service stop & start                          Example: SharpSack.exe service [service name]
DNS lookup                                    Example: SharpSack.exe dnslookup [hostname]
Deserialize input string                      Example: SharpSack.exe deserialize [string]
Convert to base64                             Example: SharpSack.exe tobase64 [string or filename]
Convert from base64                           Example: SharpSack.exe frombase64 [string or filename]
Start process as user                         Example: SharpSack.exe processasuser [username] [password]
Get directory file count                      Example: SharpSack.exe filecount [directory]
Convert file to array string                  Example: SharpSack.exe filetoarray [filename]
List directory contents                       Example: SharpSack.exe dir {path} {"recurse"}
Get environment info                          Example: SharpSack.exe whoami
Get environment info                          Example: SharpSack.exe hostname
Create a directory                            Example: SharpSack.exe mkdir [directory name]
Check network connections                     Example: SharpSack.exe netstat
Get installed programs                        Example: SharpSack.exe programs
Get SSL certificate details                   Example: SharpSack.exe ssldetails [url]
*DEV* View remote desktop sessions            Example: SharpSack.exe rdsessions {server}
View user sessions                            Example: SharpSack.exe sessions
Get network information                       Example: SharpSack.exe ipconfig
Do a reverse DNS lookup                       Example: SharpSack.exe rdns {host1,host2}
Launches a hidden instance of IE via COM      Example: SharpSack.exe hiddenie
Checks file permissions                       Example: SharpSack.exe perms {file}
Enumerate environment variables               Example: SharpSack.exe env
Create a process that just hangs out          Example: SharpSack.exe inception [processname] {ppid}
Search processes                              Example: SharpSack.exe searchps [processname]
```
